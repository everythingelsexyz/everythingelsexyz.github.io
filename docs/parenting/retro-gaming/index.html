Knulli stuff

Can sort by game types when searching

in game Hotkeys

save state: menu + y load state: menu + x retroarch menu: menu + b fast forward: menu + dpad right dpad to analog toggle: menu + select turn off power lef: menu + power exit game: menu + start

Setup stuffs:

turn off on screen help settings > user interface settings

turn on show favorites at top settings > user interface settings

settings > Game collection settings: turn off commodore 64, emulators, all the ports but “ports”, pygame, tools, od-commander

game settings:

turn on auto save/load incremental savestates (increment per save) turn on toggle fast forward turn on retro acheivements

can switch confirm and cancel buttons under development settings

Per system settings:

GBA in GBA menu, press select > advanced system options > decorations > decoration set > default knulli game rendering and shaders > scanlines shaders set

when you have a game selected, you can hold down the confirm button to get the options for that specific game

Scraping

menu > scraper create a screenscraper account online. Limited amount of scrapes per day unless you sign up to their patreon Change the scraper region to USA

make sure scrape video is off.

Can select the systems to scrape in the scraping settings as well.

By default it only scrapes stuff that hasn’t been scraped recently.

Can scrape games 1 by 1 by holding select on the game and scraaping

game manuals

turn on manuals in screen scraper settings after the original scrape. Then run scraper on the game you want the manual for. Can view the manual from the game by holding down select when the game is highlighted.
